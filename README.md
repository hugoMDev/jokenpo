# 🪨 📄 ✂️ Jokenpô (Pedra, Papel ou Tesoura)

Um jogo clássico de Jokenpô desenvolvido em ambiente web utilizando HTML, CSS e JavaScript Vanilla. O projeto simula a disputa tradicional contra a inteligência artificial (computador).

> ⚠️ **Status do Projeto:** 🛠️ Em desenvolvimento / Em construção.

---

## 🚀 Funcionalidades Planejadas

- [x] Interface básica estruturada em HTML.
- [x] Estilização e alinhamento dos elementos com CSS (Flexbox).
- [x] Seleção de jogada pelo usuário com feedback visual (opacidade) via JavaScript.
- [ ] Lógica de escolha aleatória para o oponente (computador).
- [ ] Verificação e validação do vencedor de cada rodada.
- [ ] Sistema de pontuação/placar em tempo real.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estruturação das opções do jogador e do inimigo.
- **CSS3:** Alinhamento responsivo e estilização de elementos.
- **JavaScript (ES6):** Manipulação do DOM e lógica de eventos de clique.

---

## 📦 Como Executar o Projeto

Como o projeto utiliza apenas tecnologias front-end estáticas, você não precisa instalar dependências pesadas.

1. Clone este repositório:
   ```bash
   git clone https://github.com
   ```
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` diretamente em qualquer navegador web de sua preferência.
